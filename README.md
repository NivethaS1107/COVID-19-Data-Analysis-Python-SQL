# COVID-19 Data Analysis Using Python and SQL

## Project Overview

This project analyzes real-world COVID-19 data using Python, MySQL, and interactive visualizations.

The project focuses on:
- COVID case analysis
- death analysis
- vaccination analysis
- trend analysis
- country performance ranking
- population density analysis

Interactive charts were created using Plotly for better data exploration.

---

## Technologies Used

- Python
- Pandas
- NumPy
- MySQL
- SQLAlchemy
- Plotly
- Scikit-learn
- Jupyter Notebook

---

## Key Business Problems Solved

1. Top countries by COVID cases
2. Top countries by COVID deaths
3. Countries with highest vaccination coverage
4. Countries with highest death rate percentage
5. GDP vs COVID cases analysis
6. GDP vs COVID deaths analysis
7. Comparative country-level analysis
8. Best performing countries during COVID
9. Global COVID trend analysis
10. Vaccination vs death relationship
11. Continent-wise COVID analysis
12. Population density vs COVID spread

---

## Key Insights

- Vaccination helped reduce severe COVID impact.
- Multiple COVID waves were identified globally.
- Population density influenced transmission levels.
- Some countries managed the pandemic effectively through strong vaccination coverage and lower mortality.
